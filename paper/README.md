# Paper

This directory contains some supplementary material for the paper *Simple generators of rational function fields*, 2026 (URL: ...).
We used Julia 1.12.4, Singular 4.4.1, and Maple 2025.

## Data

`data` directory. The benchmark suite from the paper. Contains problems originating from structural parameter identifiability. For each problem, it includes: the original generating set, the output of our algorithm, and the Julia script to run our algorithm.

## Examples

`examples` directory. Source code for examples from the paper.

## Scripts

`scripts` directory. Scripts for producing tables from the paper.
