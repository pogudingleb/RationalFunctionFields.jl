## Section 7.3: Fields of invariants

Two files correspond to each example:

 * an `*.mpl` file producing the original field generators using AIDA package (https://www-sop.inria.fr/members/Evelyne.Hubert/aida/) written in Maple;

 * a `*.jl` file running simplification for these generators (already in the script, no need to run the Maple).
